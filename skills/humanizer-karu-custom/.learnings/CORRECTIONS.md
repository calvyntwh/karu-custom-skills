# Humanizer Corrections

Individual corrections with context - each entry documents a specific instance where the humanizer either:
- Caught a new pattern
- Made a correction that turned out to be particularly good
- Received feedback that improved the output

Format:
```markdown
## [YYYY-MM-DD] {Brief Description}

**Original phrase:** "{the AI-generated text}"
**Why flagged:** {specific pattern category or "new pattern"}
**Applied fix:** "{your rewrite}"
**Context:** {where this appeared - topic, domain, tone}
**Outcome:** {what made this noteworthy - surprising fix, difficult case, etc.}
---
```
